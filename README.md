## Про приложение

Привет!
В данном приложении реализован функционал авторизации пользователя.
В случае, если пользователь не авторизован, доступ к странице с профилем - закрыт.
При введении корректных данных появляется аватар (при нажатии на него - можно выйти), и происходит переадресация на страницу с профилем.
Данные для проверки:

```sh
Логин - Admin
Пароль - 12345
```

Описание маршрутов:

- / — главная
- /login — страница ввода логина и пароля
- /profile — страница с профилем


Реализовано при помощи функциональных компонентов React. 
Для стилизации элементов использовались компоненты и стили Material UI.

## Запуск

Чтобы запустить проект, выполните следующие шаги:

1. Склонируйте репозиторий
   ```sh
   git clone git@github.com:AnastasiaKhot/Auth-form.git
   ```
2. Установите зависимости
   ```sh
   npm i
   ```
3. Запустите проект в режиме разработки
   ```sh
   npm run dev
   ```
